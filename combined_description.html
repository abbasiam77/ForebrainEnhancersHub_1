Combined Description for Forebrain Enhancers

1. Forebrain Enhancers

This track includes a genome-wide catalogue of predicted, 25,000 human forebrain enhancers. These enhancers are associated with human forebrain development and gene regulation.

To learn more about this dataset, please visit the full description: Forebrain Enhancers Description.

2. HES5-FOXP2-GATA3 Enhancers

This track includes predicted, 2614 forebrain enhancers, with HES5-FOXP2-GATA3 binding syntax, which plays a critical role in brain development.

To learn more about this dataset, please visit the full description: HES5-FOXP2-GATA3 Enhancers Description.
